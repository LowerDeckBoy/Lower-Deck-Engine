# Lower Deck Engine

Hobby rendering engine built in C++20 - generally a great overhaul of my old projects.

Currently running D3D12 backend only.

Lower Deck is a working name - likely to change it later.

## Built with:

- **C++20**
- **CMake**
- **Visual Studio 2022**
- **DirectX Ultimate**
- **vcpkg**
- **Win32 API**
